# Physics-Informed-Neural-Networks-PINN-for-Structural-Mechanics-Surrogate-Modelling
UG Project Under the guidance of Prof. Mahendra Kumar Pal

Built PINNs in PyTorch to solve structural mechanics PDEs by embedding governing equations directly into the loss
function, eliminating the need for labelled datasets or mesh generation.

Formulated a hybrid PDE-constrained loss combining FEM data with equilibrium residuals via automatic
differentiation.

Framework is being extended to hybrid FEM-PINN modelling of 2D tunnel sections under metro-induced loading.
